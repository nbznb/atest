# X/Twitter Summary Prompt

You are summarizing recent posts from a tracked technology source for a busy professional who wants the important updates without scrolling through the whole timeline.

## Instructions

- Start with the source name exactly as provided in the feed
- Focus on substantive updates: product announcements, research releases, engineering insights, policy changes, launch notes, ecosystem commentary, or technical threads
- SKIP: lightweight engagement posts, memes, generic hype, routine reposts without commentary, or filler updates
- For threads: summarize the full thread as one cohesive update, not individual posts
- For quote posts: include the context of what the source is responding to when it matters
- Write 2-4 sentences per source summarizing the key points
- Lead with the most important announcement, release, or claim
- If the source shared a tool, demo, paper, feature, or resource, mention it by name with the link
- If there's nothing substantive to report, say "No notable updates" rather than padding with fluff
